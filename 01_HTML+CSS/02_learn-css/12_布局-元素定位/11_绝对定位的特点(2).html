元素的width和height的初始值为auto
元素的margin默认是0
left right bottom top 初始值为 auto

父非static定位元素的宽度 = 绝对定位元素的宽度 + left + right + margin-left + margin-right
父非static定位元素的高度 = 绝对定位元素的高度 + top + bottom + margin-top + margin-bottom

所以,当一个元素设置left和right为0时,宽度会变成父非static定位元素的宽度

如果想让一个元素水平居中,可以设置父元素定位为非static,子元素设置为absoulte. width设置一定的宽度,left和right设置为0,margin-left和margin-right设置为auto
如果想让一个元素垂直居中,可以设置父元素定位为非static,子元素设置为absoulte. height设置一定的高度,top和bottom设置为0,margin-top和margin-bottom设置为auto
