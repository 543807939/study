开启了flex布局的容器叫做: flex container
flex container里的直接子元素叫做 flex item

flex item的布局会受到flex container属性的设置来进行控制和布局
flex item 不再严格区分行内和块级元素
flex item 默认包裹内容,但是可以设置宽高

可以通过设置display为flex或者inline-flex来设置容器开启flex布局

元素是沿着主轴方向排序的,默认是x轴;可以通过flex-direction来改变主轴的方向
