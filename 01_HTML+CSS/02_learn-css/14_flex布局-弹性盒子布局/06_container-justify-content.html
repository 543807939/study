justify-content决定了flex item在main axis上的排序方式:
flex-start:与main start 对齐
flex-end: 与main end 对齐
center: 居中对齐
space-between: flex items 之间的距离相等,与main start 和main end 两端对齐
space-around: flex items 之间的距离相等, flex item 与main start,main end之间的距离为flex items之间距离的一半
space-evenly: flex items 之间的距离相等,flex item 与 main start,main end之间的距离等于flex itsms之间的距离
