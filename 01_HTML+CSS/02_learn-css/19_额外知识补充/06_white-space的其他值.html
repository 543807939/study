white-space 可以设置文本空格和换行的处理方式
默认值是 normal: 合并连续的空白符,换行符会被当作空白符, 超出行盒会换行
nowrap: 合并连续的空白符,换行符会被当作空白符, 文本不会换行
pre: 连续空白会被保留,遇到换行符或者 <br>才会换行
pre-wrap:连续的空白符会被保留。在遇到换行符或者
元素，或者需要为了填充「行框盒子 (line boxes)」时才会换行。
pre-line: 连续的空白符会被合并。在遇到换行符或者
元素，或者需要为了填充「行框盒子 (line boxes)」时会换行。
