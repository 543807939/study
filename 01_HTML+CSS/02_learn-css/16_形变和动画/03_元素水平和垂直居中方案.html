水平居中的方法:
1. 如果是行内元素,给它的父元素设置text-align:center;
2. 如果是块级元素,给它自己设置 margin: 0 auto;
3. 绝对定位, 给父元素设置position为非static,自身设置position为absolute;设置一个宽度,left:0;right:0;margin: 0 auto;
4. 绝对定位, 给父元素设置position为非static,自身设置position为absolute;设置自身left:50%;transform:translateX(-50%);
5. flex布局,设置justify-content:center;

垂直居中的方法:
1. 绝对定位, 给父元素设置position为非static,自身设置position为absolute;设置一个宽度,top:0;bottom:0;margin: auto 0;
2. flex布局,设置align-item:center;
3. 绝对定位, 给父元素设置position为非static,自身设置position为absolute;设置自身top:50%;transform:translateY(-50%);
